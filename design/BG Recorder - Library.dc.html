Game log

{{ logConnLabel }}

Battlegrounds has no live API. The recorder tails Power.log — the game's own event log — to detect matches, turns, and combat results. Advanced logging must be enabled once.

One-time setup
  1. Click Write log.config below — it enables the Power logger in %LocalAppData%\Blizzard\Hearthstone.
  2. Restart Hearthstone if it's running.
  3. The recorder tails Logs\Power.log in the install folder below.

Takes effect the next time Hearthstone launches.

log.config written — restart Hearthstone 
D 21:44:02 PowerTaskList.DebugPrintPower() - TAG_CHANGE Entity=GameEntity tag=TURN value=9
D 21:44:02 GameState.DebugPrintPower() - BLOCK_START BlockType=ATTACK Entity=[id=214]
D 21:44:03 GameState.DebugPrintPower() - TAG_CHANGE Entity=BobsTavern tag=STEP value=MAIN_COMBAT
D 21:44:05 GameState.DebugPrintPower() - TAG_CHANGE Entity=[id=64] tag=DAMAGE value=12
